# 项目简介

忘忧草安全团队研发的分布式信息探测与脆弱性扫描系统(nstscan)的java消费端:laughing:

从消息队列(rabbitMQ)获取任务，实现漏洞检测、CMS识别、环境探测、多种协议的暴力破解:innocent:
